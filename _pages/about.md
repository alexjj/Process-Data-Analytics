### Using this site

This website is powered by [fastpages](https://github.com/fastai/fastpages). There are many [static site generators](https://www.staticgen.com/), but this one integrates [Jupyter](https://jupyter.org/) notebooks so effortlessly that it seemed perfect for this blog. Perhaps if I was 100% [R](https://www.r-project.org/) I might use [blogdown](https://bookdown.org/yihui/blogdown/).

At the end of each post is a comments section, it requires a Github account to use. If the page has been made from a jupyter notebook then the Colab, Binder and Github buttons at the top of the page allows you to see the real notebook and, in the case of Colab and Binder, interact with it.
